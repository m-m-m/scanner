The module `mmm-scanner` helps to parse character sequences efficient and easily.

=== Usage

Maven Dependency:
```xml
<dependency>
  <groupId>io.githib.m-m-m</groupId>
  <artifactId>mmm-scanner</artifactId>
</dependency>
```

Module Dependency:
```java
  requires transitive io.github.mmm.scanner;
```
